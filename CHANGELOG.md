# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-01-04
### Added
- API-first CLI for iFood with search, discovery, catalog, cart creation, and checkout review.
- Playwright-based session handling for authenticated browser context.
- Parsers for categories, merchants, and menu items.
- Basic tests and CI.
- Agent-focused skill documentation.

## [0.1.1] - 2026-01-04
### Fixed
- CI build by externalizing Playwright from the bundle and using a Node shebang for the CLI.
